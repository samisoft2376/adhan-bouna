# 🕌 أذان بونة Pro | Adhan Bouna

**تطبيق ويب تقدمي (PWA) تفاعلي لمواقيت الصلاة، يجمع بين الدقة البرمجية وفخامة التصميم العالمي.**

*تطوير وتصميم بلمسة جزائرية:* **Kherrachi Sami** 🇩🇿

---

## 🌟 لمحة عن التطبيق
"أذان بونة Pro" ليس مجرد صفحة ويب تقليدية، بل هو تطبيق متكامل (PWA) يمكن تثبيته على الهواتف الذكية. تم تصميمه بهوية بصرية فاخرة مستوحاة من الواجهات الرقمية المتقدمة (Dashboard Widgets)، ليقدم تجربة مستخدم (UX) سلسة، دقيقة، ومريحة للعين، مخصصة لمدينة عنابة والجزائر عامة.

## ✨ المميزات الرئيسية (Features)
* 📍 **تحديد الموقع الذكي (Geolocation):** جلب مواقيت الصلاة بدقة متناهية بناءً على الموقع الجغرافي الفعلي للمستخدم عبر واجهة `Aladhan API`.
* ⚡ **العمل بدون إنترنت (Offline-First):** بفضل تقنية `Service Worker`، يعمل التطبيق بكفاءة حتى عند انقطاع الاتصال بالإنترنت.
* 🎨 **واجهة ديناميكية متفاعلة (Dynamic UI):** * يتغير لون التطبيق تدريجياً إلى **الذهبي الفاخر** قبل الأذان بـ 30 دقيقة.
  * يتحول إلى **البرتقالي الملكي (Urgent)** قبل الأذان بـ 5 دقائق.
* 📱 **تطبيق قابل للتثبيت (PWA):** يمكن للمستخدمين إضافته إلى "الشاشة الرئيسية" للهاتف ليظهر ويعمل كتطبيق أصلي (Native App) بفضل ملف `manifest.json`.
* 🔔 **تنبيه صوتي للأذان:** إمكانية تفعيل الأذان الصوتي تلقائياً عند دخول الوقت.
* ⏱️ **إطار معلومات مدمج (Premium Frame):** يعرض الساعة الحية، التاريخ، والعد التنازلي الدقيق للصلاة القادمة داخل إطار زجاجي داكن.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **HTML5 / CSS3:** لبناء الهيكل وتصميم الحركات الانسيابية (Animations) والتدرجات اللونية (Gradients).
* **Vanilla JavaScript (ES6):** لبرمجة المنطق، حساب الفروق الزمنية، وإدارة الحالات الديناميكية.
* **PWA (Progressive Web App):** * `manifest.json` لإدارة الهوية البصرية للتطبيق.
  * `sw.js` (Service Worker) لإدارة الذاكرة المخبأة (Caching) والعمل دون اتصال.
* **APIs:** `Aladhan API` و `Geolocation API`.
